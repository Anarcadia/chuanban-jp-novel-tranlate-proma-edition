```yaml
- 原文: 張三
  译文: 张三
  类型: 人名
  别称:
    - 老张
  简介: 主角，男，18岁

# ═══ 术语表结束 ═══
```

## term_notes.yaml

```yaml
- 条目: 人名敬称处理

  理由: 保持文风简洁自然

  置信度: 高
```

```yaml
风格指南: |
  都市轻小说，第一人称叙事，语言轻松幽默。对话口语化，内心独白保持吐槽感。

  主角意外卷入异常事件，在日常生活与隐藏身份之间寻找平衡。

  上一章的主要情节，控制在约100字。
```
